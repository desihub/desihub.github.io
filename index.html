DESI Data Systems Organization Web Pages

Data

These packages form the data pipeline from target selection through final redshifts.

  • Target Selection
    • desitarget : Target selection based upon Legacy Survey tractor bricks.
  • Survey Planning and Operations
    • fiberassign : Fiber assignment; Python/C++.
    • desisurvey : Survey planning and implementation (e.g., next field selector).
    • desietc : Online real-time dynamic exposure time calculator.
  • Spectro Pipeline
    • desispec : Spectroscopic data processing algorithms and pipeline.
    • specex : Spectral PSF measurements (not DESI-specific).
    • specter : Spectroperfectionism extractions (not DESI-specific).
      • Also used for pixel-level simulations.
    • redrock : Classifications and redshift fitting.

Core

These are utility packages used by others.

  • desiutil : Installation and versioning support, logging, mask bit helper code, etc.
  • desimodel : Describes the key DESI hardware and survey parameters; mostly data with a little I/O helper code.
  • speclite : Spectroscopic utility routines that aren't DESI-specific.

Simulation

These packages are for simulating DESI.

  • desisim : DESI-specific simulations, including:
    • pixsim-desi : Wraps specter for pixel-level simulations.
    • quickgen: wraps quicksim (soon specsim) to quickly generate pipeline-format output files
    • quickbrick: (coming soon)
    • quickcat: (coming soon)
  • specter : Pixel-level raw data simulations (not DESI-specific).
    • Also used for spectroperfectionism extractions.
  • specsim : Quick spectrum -> spectrum simulator of throughput, noise, and resolution (not DESI-specific).
  • surveysim : Simulating nightly operations, weather, etc.

Infrastructure

  • desimodules : Set up the DESI software stack.
  • desidatamodel : Describe data files used by DESI.
  • desitemplate : Guide for creating new DESI packages.
  • desitree : Set up environment variables pointing to various parts of the DESI data directory tree.
  • desiBackup : Configure HPSS backups.
